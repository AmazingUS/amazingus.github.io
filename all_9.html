Hexo博客搭建使用

  • 一. 搭建步骤
  • 二. Hexo使用
  • 三. 备份管理
  • 四. 重新安装

一. 搭建步骤

  1. 安装Git；

  2. 安装Node.js；

  3. 通过Node.js安装hexo-cli模块（可以执行hexo命令）；

    sudo npm install hexo-cli -g

  4. Hexo创建博客；

    hexo init <BLOG_FOLDER> # 创建博客，会自动安装Hexo所需要的模块

cd <BLOG_FOLDER>
npm ls --depth=0 # 查看安装的本地模块

一、概述

  • CocoaPods是iOS、macOS等平台的第三方框架的管理工具，类似于Android中的Gradle；但是没有Gradle强大，Gradle不仅可以自动下载配置所需要的框架，还可以进行自动化构建；
  • CocoaPods基于Ruby，是Ruby的一个包（库）；需要确保使用最新的Ruby/Gem进行安装CocoaPods，或者直接安装CocoaPods APP；
