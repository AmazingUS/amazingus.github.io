Redis配置使用（Linux）

2018/8/28
  • 一、概述、安装
  • 二、Redis命令行客户端

一、概述、安装

  • Redis是一个key-value存储系统，与和Memcached类似；
    • 其支持存储的value类型有string(字符串)、list(链表)、set(集合)、zset(有序集合)等，并且对这些数据类型支持原子性的push/pop、add/remove、交并和差集等操作；
    • 为了保证效率，数据先缓存在内存中，之后会周期性地将更新的数据写入磁盘或把修改操作写入追加的记录文件中，进而可以实现了master-slave(主从)同步；

  • 安装

    apt-get install redis-server
// 配置文件为/etc/redis/redis.conf

systemctl status|restart redis.service

二、Redis命令行客户端

  • 安装Redis服务器（redis-server）时，会自动安装Redis命令行客户端程序；在本机终端中输入redis-cli命令就可以启动，访问Redis服务器；

    redis-cli

# 查看所有的key列表
keys *

# 增加记录key1
set key1 "hello"
# 打印记录key1
get key1
# 删除记录key1
del key1

# 增加一个列表记录key3
lpush key3 a
# 从左边插入列表
lpush key3 b
# 从右边插入列表
rpush key3 c
# 打印列表记录（从左到右）
lrange key3 0 2

  • 默认情况下，访问Redis服务器是不需要密码的；修改配置文件redis.conf，取消注释requirepass设置访问密码；

    root@ubuntu:~# redis-cli
127.0.0.1:6379> keys *
(error) NOAUTH Authentication required.

redis-cli -a mypass # 添加密码参数方法
root@ubuntu:~# redis-cli -a mypass
127.0.0.1:6379> keys *
1) "key1"

  • 默认情况下，Redis服务器不允许远程访问，只允许本机访问；修改配置文件redis.conf，注释掉bind 127.0.0.1 ::1打开远程访问；

    redis-cli -a mypass -h 192.168.1.100
