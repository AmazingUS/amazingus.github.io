本地多个SSH Key配置使用

  • 一、概述
  • 二、使用多个SSH Key
  • 三、.ssh/config文件

一、概述

  • 执行 ssh-keygen -t rsa -C "xx@email.com"，回车确认，默认在~/.ssh/目录中生成名称为“id_rsa”的private key和“id_rsa_pub”的public key；注意：每次执行生成的key是不一样的；
  • 对于不同的网站（如github.com、gitee.com等），虽然是不同的账号，一般使用同一个key进行认证，也可以使用不同的key（配置.ssh/config）；
  • 对于同一网站，若需要使用不同的账号，就需要配置不同的key（配置.ssh/config），同一个key只能用于一个账号；

一、安装

  • Ruby安装方式有多种方式，可以通Ruby安装包、Homebrew、rbenv或RVM等方式；
  • rbenv和RVM都可进行Ruby多版本管理，等同于Node.js中的nvm、Python中的pyenv；这里使用rbenv进行安装（rbenv的GitHub Star比RVM的多）；

    • 第1步：安装rbenv

      brew install rbenv

    • 第2步：安装某个版本的Ruby；

      rbenv install <VERSION>
