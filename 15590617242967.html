Sublime配置使用

2019/5/29
  • 一、个人配置
  • 二、插件安装
  • 三、快捷键

一、个人配置

  • Preference -> Package Setting -> Package Control -> Setting-User

  • 1、添加以下配置，设置代理：

    "http_proxy": "http://127.0.0.1:8001",
"https_proxy": "http://127.0.0.1:8001",

二、插件安装

  • 1、安装Package Control（安装其他插件的必要基础）

    • 安装方式：自动安装、手动安装；
    • 安装文档：https://packagecontrol.io/installation；

  • 2、安装JsFormat（HTML/CSS/JS/JS代码格式化插件）

    • 系统默认的Edit -> Line -> Reindent，只是对当前的文本进行重新缩进（需要先全选文本，再执行操作）；
    • 调出输入框命令面板，输入install选择Package Control: Install package，之后输入选择JsFormat进行安装；
    • 使用：调出输入框命令面板，再输入Format: Javascript就进行了格式化命令；

三、快捷键

  • Ctrl + `：显示底部console；
  • CMD + Shift + P：调出输入框命令面板；
  • CMD + J：多行文件合并到一行；
