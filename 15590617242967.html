Sublime配置使用

2019/5/29
  • 一、个人配置
  • 二、插件安装
  • 三、快捷键

一、个人配置

  • Preference -> Package Setting -> Package Control -> Setting-User

  • 1、添加以下配置，设置代理：

    "http_proxy": "http://127.0.0.1:8001",
"https_proxy": "http://127.0.0.1:8001",

二、插件安装

  • 1、Package Control（安装其他插件的必要基础）

    • 安装方式：自动安装、手动安装；
    • 安装文档：https://packagecontrol.io/installation；

  • 2、JsFormat（HTML/CSS/JS/JS代码格式化插件）

    • 系统默认的Edit -> Line -> Reindent，只是对当前的文本进行重新缩进（需要先全选文本，再执行操作）；
    • 调出输入框命令面板，输入install选择Package Control: Install Package，之后输入选择JsFormat进行安装；
    • 使用：调出输入框命令面板，再输入Format: Javascript就进行了格式化命令；

  • 3、Color Highlight（RGB等文字实时渲染显示颜色效果插件）

  • 4、Color Convert（颜色格式转换插件）

  • 5、ConvertToUTF8、Codecs33（解决中文GBK乱码插件）

三、快捷键

  • 显示底部console：Ctrl + ` ；
  • 调出输入框命令面板：CMD + Shift + P；
  • 多行文件合并到一行：CMD + J；
  • 多行编辑：
    • 鼠标框选多行后，CMD + Shift + L：进入多行编辑模式；
    • CMD + -> 或 Ctrl + A：移到行首编辑；
    • CMD + <- 或 Ctrl + E：移到行末编辑；
