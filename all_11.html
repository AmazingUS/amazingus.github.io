  • 三、备份管理

一、搭建步骤

  • 1、安装Git；

  • 2、安装Node.js；

  • 3、通过Node.js安装hexo-cli模块（这样就可以执行hexo命令了）；

    sudo npm install hexo-cli -g

  • 4、创建Hexo博客；

    hexo init <BLOG_FOLDER> # 创建博客，会自动安装Hexo所需要的模块

cd <BLOG_FOLDER>
npm ls --depth=0 # 查看安装的本地模块

  • CocoaPods基于Ruby，是Ruby的一个包（库）；需要确保使用最新的Ruby/Gem进行安装CocoaPods，或者直接安装CocoaPods APP（已废弃）；
