01. 简介

1.1 Python模式
  • 交互模式（终端输入“python3”进入交互模式，输入“exit()”退出）
  • 命令行模式（终端输入“python3 文件名.py”直接运行）
1.2 输入、输出函数
  • 输出函数print(a1, a2, ...)，输出参数间遇到一个“,”就会拼接一个空格符，例如print('hello','world')结果为“hello world”
  • 输入函数input()，函数返回值为输入数据，例如name = input()、 带提示信息的输入name = input('please enter your name:')
1.3 细节
  • Python是区分大小写的
  • 注释是以#开头
  • 一行Python代码通常为一条语句；当语句以“:”结尾时，之后下面缩进的代码为一个代码块，相当于其他语言中的大括号中的代码；建议使用4个空格进行缩进
1.4 字符转义
  • “\”表示转义，例如print('I\'m learning\nPython.')
  • 若字符串中的所有特殊字符都需要转义，则使用r''，例如print(r'\tabc\r\'') # 结果为\tabc\r\'
1.5 换行
  • 若字符串中需要输出换行，可以使用\n，也可以使用'''abc'''的形式输入（此时输出也会是换行）

  • '''abc'''的形式输入：使用3个“'”代替1个“'”，回车换行时“>>>”变成“...”，表示接着上一行内容输入（输入内容被添加了换行符），可以多次回车换行，输入完毕以3个“'”结束

    >>> s = '''a 
... b  
... c  
...   
... '''  
>>> s  
'a\nb\nc\n\n'

  • 注意：无论是在' '形式，还是在''' '''形式的输入过程中，语句末尾使用“\”都只是显示换行（防止单行书写过多，导致阅读代码不便）

    >>> s1 = 'a'\
... 'b'\
... 'c'
>>> s1
'abc'

>>> s2 = '''a \
... b \
... c
... 
... '''
>>> s2
'a b c\n\n'

02. 基础

2.1 数据类型、变量
  • 布尔值：False、True；布尔运算：and、or、not；空值：使用None表示该特殊值
  • Python中变量名都是一个引用
  • Python中通常使用全部大写的变量名表示常量（仅仅是命名规范，值还是可以被修改）
  • Python是动态语言，a = 10之后还可以进行a = "ABC"
  • 除法：/结果为浮点数、//结果为整数（地板除，取整）、%结果为余数
2.2 字符编码
  • ASCII编码为一个字节（表示数字字符、英文字符、特殊字符），GB2312编码为两个字节（表示中文字符）
  • 为了统一包含所有的语言字符创造了Unicode编码（一个字符使用两个字节，高版本Unicode为3个字节；ASCII中的字符在Unicode中通过前一个字节补0表示）
  • 若文本中都是英文字符（ASCII中的字符），则使用Unicode编码浪费一半的字节，故创造了“可变长编码”UTF-8
  • 在UTF-8编码中，英文字符为1个字节，中文字符为3个字节，生僻字符为4~6个字节；可以认为ASCII编码是UTF-8编码的一部分
  • 计算机内存中统一使用Unicode编码，当需要保存到硬盘或传输时才转为UTF-8编码或其他编码（为了节约存储）；Python3中的字符为Unicode编码（Python2不是）
  • ord()函数：获取单个字符对应的10进制的Unicode编码值，例如ord('A')为65、ord('中')为20013
  • chr()函数：将10进制的Unicode编码值转换为对应字符，例如chr(66)为'B'、chr(20013)为'中'
  • Python中表示字符：
    • a. 直接表示，如'A'、'中'
    • b. 加\u前缀的16进制的的特殊字符串（Unicode编码值），例如'\u4e2d'为'中'，4e2d的16进制就是10进制的20013
      • a) 必须16进制
      • b) 必须加\u前缀，否则和普通字符串没有区别
2.3 字符串
  • Python中的字符串类型为str；在内存中以Unicode表示，若保存到硬盘或传输时需将str进行编码，变成以字节为单位的bytes
  • Python中的bytes类型的数据（单、多字节）是加“b”前缀的16进制的特殊字符串（不加“b”前缀就是字符串了），例如x = b'ABC'，输出打印为b'ABC'
  • bytes类型的数据输出打印时，无法显示为ASCII字符的字节使用b'\x##'表示

  • encode()函数：将str转为指定编码的bytes（注意中文无法使用ASCII编码）

    'ABC'.encode('ascii') # 输出打印为b'ABC'
'ABC'.encode('utf-8') # 输出打印为b'ABC'
'中文'.encode('utf-8') # 输出打印为b'\xe4\xb8\xad\xe6\x96\x87'

  • decode()函数：将bytes转为str，例如b'\xe4\xb8\xad\xe6\x96\x87'.decode('utf-8')结果为'中文'

  • len()函数可计算str的字符数，也可计算bytes的字节数，例如len('中文')为2、len('中文'.encode('utf-8'))为6

  • 为了让Python源代码文件被读取时是以UTF-8编码读取，应在最前面添加注释 # -*- codeing: utf-8 -*- ，同时保存Python源代码文件时必须以“UTF-8 without BOM”编码保存

2.4 str格式化

  • 参数使用%s、%d、%f、%x(十六进制)等代替，格式化语句与实际值之间使用%分割，例如

    'Hello, %s' % 'world'
'%2d-%02d' % (3, 1)
'growth rate: %d %%' % 7 

s = 'world'
print('hello', s)           # hello world
print('hello %s' % s)       # hello world
'hello %s' % s              # 'hello world'
print('hello %s' % s, s)    # hello world world
2.5 str.format()方法

  • format()方法会返回一个新的字符串；在新的字符串中，原字符串的“替换字段（大括号及其中的数字等为替换字段）”被方法中的参数所代替

    # 按照位置
>>> '{0}, {1}'.format('tom', 123)
'tom, 123'
>>> '{}, {}'.format('tom', 123)
'tom, 123'
>>> '{1}, {0}, {1}'.format('tom', 123)
'123, tom, 123'
>>> '{{{1}}}'.format('tom', 123) # 两个{转义后一个{
'{123}'
>>> '{{1}}'.format('tom', 123)
'{1}'

# 按照字段名
>>> '{name}\'age is {age}'.format(name='Tom', age=123)
"Tom'age is 123"
>>> '{name}\'weight is {0},{1}'.format(123, 456, name='Tom')  # 可变参数(123，456)要放在关键字参数(name)之前
"Tom'age is 123,456"

# 按照索引
>>> names = ['Tom', 'Jack']
>>> ages = [123, 456, 789, 111]
>>> '{0[1]}\'age is {1[2]}'.format(names, ages) # 数组
"Jack'age is 789"
>>> person = {'name': 'Tom', 'age': 123}
>>> '{0[name]}\'age is {0[age]}'.format(person) # 字典
"Tom'age is 123"

# 按照对象
>>> import math
>>> 'math.pi = {0.pi}'.format(math)
'math.pi = 3.141592653589793'

# 格式化（“:”之后为限定格式）
>>> '{0:b}'.format(16)  # 转为二进制
'10000'
>>> '{0:x}'.format(16)  # 转为十六进制
'10'
2.6 list类
  • 数组，有序可变，同一个数组中的元素的数据类型可以不同（元素可以同时是'ABC'、123、True、list），
  • classmates = ['Tom', 'Jack', 123, True]创建一个list对象
  • classmates[0]访问第一个元素，classmates[-1]访问最后一个元素，s[1][0]访问s的第二个元素（list）中的第一个元素
  • classmates.append("ABC")追加一个元素到尾部，classmates.insert(1, "ABC")将元素插入到指定位置
  • classmates.pop()删除最后一个元素，classmates.pop(1)删除第二个元素
  • classmates[1] = 'TEST'替换第二个元素
  • len(classmates)计算数组的元素个数，l = []表示l是一个空数组
2.7 tuple类
  • 元组，有序不可变，同一个元组中的元素的数据类型可以不同
  • 只能访问不能修改，没有append、insert、pop等修改方法，定义tuple对象时元素就已确定
  • classmates = ('Tom', "JACK", 123)创建一个tuple对象
  • t = ()定义一个空元组，t = (1,)定义一个只有一个元素的元组（为避歧义必须增加一个逗号，否则认为是数学计算结果）
  • 元组不可被修改是指某个位置索引不能再指向其他对象，但该位置索引所指向的对象本身是可以被修改的，例如t = ('a', 'b', ['A', 'B'])之后t[2][0] = 'X'进行修改
2.8 条件判断、循环

  • 注意要写冒号“:”，缩进表示代码块“{}”；elif是“else if”的缩写

    age = 0

if age > 18:
    print('your age is', age)
   print('adult')
print('haha')

  • input()函数返回的数据类型是str，数值判断时必须进行类型转换，例如使用int()函数进行转换 if int(age) < 20:

  • range()函数可以生产一个整数“序列”，再通过list()函数转为list，例如list(range(5))产生数组[0, 1, 2, 3, 4]

  • Python中的类型转换函数，可以认为就是构造函数

  • 循环方式：for x in arr:、while bool:

2.9 dict类
  • 根据key找value：查找插入速度快，不会随key的增加而变慢；占用内存多，是一种空间换时间的方法
  • 字典，无序可变（key的类型可以不同，value亦然）
  • dict的key是唯一且不可变，且key对应的value也是唯一的
  • dict根据key通过Hash计算value的内存位置，字符串、整数都可作为key，list不可作为key
  • d = {'Tom': 95, 'Bob': 75, 'Jack': 85}创建一个dict，使用len()获取dict的key-value对数
  • d['Tom']访问'Tom'对应的value；若不存在此key则报KeyError，可先通过'Tom' in d的方式判断key是否存在，再取值
  • 也可以通过d.get('Tom')方法取值（不存在返回None；交互命令行None不显示结果），同时也可以在key不存在时赋默认值：d.get('Tom', 'haha')
  • d.pop('Tom')根据key删除key-value对并返回对应的value
  • d['abc'] = 67添加或重新赋值key-value对
2.10 set类
  • 由key（类型可以不同）组成的无序唯一的集合（可以认为是只含有key的“特殊”dict）
  • 集合中不可存放list对象，但可存放tuple对象（含有list对象的tuple对象也不可存放）
  • 创建：s = set([1, 2, 3])； 添加：s.add('a')，删除：s.remove('a')，交集：s1 & s2，并集：s1 | s2
2.11 可变、不可变对象

  • 调用不可变对象的任何方法，不会改变对象自身的内容，方法会创建新的对象并进行返回

    # 可变：如list
a = ['d', 'b', 'c'] 
a.sort()
a  # ['b', 'c', 'd']

# 不可变：如str
a = 'abc'  # a是变量，'abc'是字符串对象，a指向'abc'
a.replace('a', 'A')  # 'Abc'
a   # 'abc'
